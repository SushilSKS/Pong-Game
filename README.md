# Pong Game 🏓

A simple pong game where two players can compete each other.

## Features
- We can control the game using keyboards
- Two players can play together and compete each other
- Speed of the ball will keep getting increasing
- Scores will be tracked in scoreboard

## Controls
- Left Paddle: W (up), S (down)
- Right Paddle: Up Arrow, Down Arrow

## How to Run
1. Make sure Python is installed
2. Run:
   python main.py

## Project Structure
- main.py → game loop
- paddle.py → paddle logic
- ball.py → ball movement & physics
- scoreboard.py → score tracking

## Tech Used
- Python
- Turtle Graphics
